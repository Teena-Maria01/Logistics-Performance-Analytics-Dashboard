This project is an end-to-end Power BI dashboard developed to monitor and analyze logistics operations across hubs, drivers, and vehicles. The dashboard provides operational insights into delivery performance, hub efficiency, driver productivity, and vehicle utilization.

The objective was to transform raw logistics data into an interactive decision-support system enabling stakeholders to:

Track delivery KPIs
Monitor on-time performance
Analyze hub processing efficiency
Evaluate driver performance
Optimize vehicle utilization

The solution was built using a star schema data model and advanced DAX measures to ensure scalability, accuracy, and analytical depth.

Business Problem

Logistics companies generate large volumes of operational data across hubs, drivers, and fleet operations. However, without structured analytics, decision-makers lack visibility into:

Delivery delays
Hub performance inefficiencies
Driver productivity trends
Fleet utilization patterns

The goal of this project was to build a centralized analytics dashboard to provide real-time operational insights and support data-driven decision-making.

Key Insights Generated

Identified hubs operating below capacity utilization.
Detected drivers with consistent high delay percentages.
Analyzed relationship between vehicle age and breakdown frequency.
Observed seasonal delivery fluctuations across months.
Evaluated fleet composition impact on total orders.

Business Impact

This dashboard enables logistics managers to:

Reduce delivery delays
